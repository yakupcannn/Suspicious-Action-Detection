# Suspicious-Action-Detection

## Project Details
In this project, we focused on how to detect suspious thief actions. To do that, Media pipe and YOLO model have been used. Media Pipe library helped us for pose estimation.Using this pose estimation, we defined some suspicious actions such as lifting up, putting your hands to the pocket and crouching. This actions have told us we detect suspicious situations. Based on that , we can take actions. When it comes out the YOLO model, this helped us detecting the suspicious person.

### Used Libraries 
- Open Cv
- Media Pipe
- Ulytralytics (for YOLO model)
(You can check the library details from requirements.txt)
